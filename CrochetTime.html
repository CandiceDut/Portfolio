Crochet'Time

Etat : Finis en Octobre 2024

Ce projet est un site web dédié à la vente de doudous en crochet faits main, mettant en valeur l’artisanat et la personnalisation des produits. C'est une première version d'un futur site web qui me permettra de vendre mes créations partout en France (voire même dans le monde).

Développé en PHP, HTML et JavaScript, ce site a pour but de proposer une expérience utilisateur fluide tout en offrant une gestion simple et efficace du contenu pour l’administrateur. On peut donc y retrouver un catalogue de doudous interactif, un système de panier et de commandes ainsi que tout un back office administrateur afin de faciliter l’utilisation du site pour le vendeur.

Le site est entièrement responsive et s’adapte parfaitement aux différentes tailles d’écrans, assurant une accessibilité optimale.

Technologies utilisées :

Frontend : HTML et CSS pour la structure et le design, JavaScript pour les interactions dynamiques.
Backend : PHP pour la gestion des données et des fonctionnalités du site.
Base de données : SQL pour stocker les informations des produits, des commandes et des utilisateurs.
Outils : Visual Studio Code pour coder le site et PHPMyAdmin pour gérer la base de données.
