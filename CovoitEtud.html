COVOIT'ETUD

Etat : En cours

Ce projet est un site web dédié au covoiturage entre étudiants, permettant aux étudiants qui ne possèdent pas de voitures de pouvoir faire un trajet simple, peu cher, en bonne compagnie et en évitant les bus souvent en retard mais aussi aux étudiants qui possèdent une voiture de pouvoir réduire le cout de l'essence et de ne pas se retrouver seul lors de leurs trajets.

Développé en PHP, HTML et JavaScript, ce site a pour but de proposer une expérience utilisateur fluide et conviviale tout en offrant une gestion efficace du contenu pour l’administrateur. On peut donc y retrouver un système d'ajout, de recherche, de modification et de suppression d'un trajet mais aussi un profil utilisateur modifiable, un système d'avis portant sur un conducteur par rapport à un trajet ou encore un système de badges attribué sous certaines conditions.

Le site est entièrement responsive et s’adapte parfaitement aux différentes tailles d’écrans, assurant une accessibilité optimale.

Technologies utilisées :

Frontend : HTML et CSS pour la structure et le design, JavaScript pour les interactions dynamiques.
Backend : PHP pour la gestion des données et des fonctionnalités du site.
Base de données : SQL pour stocker les informations des trajets, des avis, des badges et des utilisateurs.
Framework : Twig avec le modèle MVC (Modèle Vue Controleur).
Outils : Visual Studio Code pour coder le site et PHPMyAdmin pour gérer la base de données.
